# Web Technologies Lab (CS 224L)

This repository serves as a personal portfolio and lab exercise collection developed for the Web Technologies course at the University of Engineering & Technology, Peshawar.

## Project Overview
This project demonstrates foundational web development skills including semantic HTML, table layouts, form integration, and responsive navigation.

## Pages Included
- `index.html`: The main landing page acting as a portfolio hub.
- `profile.html`: Personal profile page containing contact details.
- `schedule.html`: A detailed table representing the weekly course schedule.
- `registration.html`: A functional student registration form integrated with Formspree.
- `404.html`: A custom error page for handling invalid routes.

## Features
- **Semantic Navigation**: Consistent header across all pages to ensure easy site traversal.
- **Form Handling**: Integrated submission via Formspree for the registration form.
- **Responsive Design**: Includes viewport meta tags for mobile-friendly scaling.
- **Performance**: Utilizes `loading="lazy"` for image optimization.
